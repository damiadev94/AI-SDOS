# tasks/templates

Plantilla oficial para crear nuevas Tasks.

Toda nueva Task debe crearse a partir de `task-template.md`, que define la estructura mínima obligatoria (Objetivo, Tipo de Task, Entradas requeridas, Dependencias, Artefactos de salida, Procedimiento, Reglas obligatorias, Restricciones, Criterios de aceptación, Estado final, Observaciones, Historial). Ver `tasks/README.md`.
