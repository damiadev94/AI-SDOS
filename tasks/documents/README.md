# tasks/documents

Categoría reservada.

Hoy todas las Documentation Tasks del proyecto viven dentro de `tasks/architecture/` (por ejemplo `TASK-ARCH-011-generate-decision-traceability.md`), porque cada una produce un documento de arquitectura.

Esta carpeta queda reservada para futuras Documentation Tasks que no pertenezcan a la categoría `architecture` (por ejemplo, un Glossary o un ADR que no sea parte de una vista arquitectónica).

Ver [`tasks/README.md`](../README.md) para la convención completa.
