# docs/architecture/diagrams

Diagramas de las vistas de arquitectura. Inventario completo con el documento asociado a cada uno: [`architecture-index.md`](../architecture-index.md), sección 3.

`architecture-overview.html` es el único diagrama en formato HTML/SVG (Documentation Framework DDS); consume los estilos de [`assets/css/`](../../../assets/README.md) mediante rutas relativas (`../../../assets/css/*.css`) — no cambiar la profundidad de esta carpeta sin actualizar esas rutas. El resto de los diagramas son Mermaid (`.mmd`).

Diagramas de secuencia: [`sequences/`](sequences/README.md).
